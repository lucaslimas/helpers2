# ESLint

## Instalando dependências em modo de desenvolvimento

```bash
yarn add eslint -D
```

Inicie o eslint
```bash
yarn eslint --init
```

Adicione o arquivo **.eslintignore**
```
/*.js
node_modules
dist
```
